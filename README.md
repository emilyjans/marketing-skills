# marketing-skills

The community skill list behind [mktgskills.ai](https://mktgskills.ai) — 
the marketing skills directory for AI tools.

Marketing skills are markdown files that give your AI expert knowledge 
before it helps you. Install one and your AI instantly thinks like a 
specialist — SEO auditor, copywriter, CRO expert — without you explaining 
the basics every session. Works with Claude Code, Cursor, Lovable, ChatGPT, 
Perplexity, and any tool that reads markdown.

## Install a skill

```bash
npx skills add coreyhaines31/marketingskills/seo-audit
```

Find the install command for any skill at [mktgskills.ai](https://mktgskills.ai).
