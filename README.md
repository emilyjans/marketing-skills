# mktgskills-directory

The community skill list behind [mktgskills.ai](https://mktgskills.ai) — 
the marketing skills directory for AI tools.

## What is a skill?

A skill is a markdown file that gives your AI expert marketing knowledge 
before it helps you. Install one and your AI instantly thinks like a 
specialist — SEO auditor, copywriter, CRO expert — without you explaining 
the basics every session.

## Submit a skill

Have a marketing skill to share? Open an issue using the 
**[Submit a Skill](../../issues/new?template=submit-skill.yml)** template.

We accept:
- **Free skills** — public GitHub repos with a SKILL.md file anyone can install
- **Paid skills** — purchase or subscription pages with real marketing content

Approved skills are added to the directory at mktgskills.ai within a few days.

## Edit an existing listing

Found a broken link, wrong category, or outdated description?
Open an **[Edit Request](../../issues/new?template=edit-skill.yml)**.
